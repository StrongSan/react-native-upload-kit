# Contributing

Thanks for considering a contribution.

## Development

```sh
npm install
npm test
npm run typecheck
npm run build
```

## Pull Requests

- Keep changes focused on React Native upload behavior.
- Add tests for new normalization, validation, multipart, or provider behavior.
- Avoid committing secrets, real API keys, access tokens, or production URLs.
- Document platform-specific behavior when it differs between Android and iOS.

## Good First Issues

Useful early contributions include:

- More MIME type mappings.
- Expo ImagePicker examples.
- S3, R2, or Firebase Storage examples.
- Backend snippets for Express, NestJS, Rails, Laravel, or Django.
